================================================================================
                          8. DISCOUNT SYSTEM
================================================================================

Overview:
---------
The discount system uses a two-layer calculation approach with automatic 
override capability. This ensures customers always see accurate discount 
percentages based on real price differences.

Layer 1: Product Loading (Data Processing)
------------------------------------------
When products are loaded from Google Sheets, the system calculates the 
discount percentage automatically using the formula:

    Discount % = (Old Price - Current Price) / Old Price × 100

This calculated value becomes the product's final discount, regardless of 
what discount value exists in the sheet.

Console Warning System:
-----------------------
If the sheet's discount percentage doesn't match the calculated value, 
a warning appears in the console:

    "Discount in sheet (10%) doesn't match calculation based on 
    Rs.900 vs Rs.1000. Updated to 10%."

Implementation (main.js:918-928):
---------------------------------
```javascript
const oldPrice = parseFloat(row['Old Price'] || 0) || null;
const sheetDiscount = parseFloat(row['Discount %'] || 0) || 0;
let calculatedDiscount = 0;

if (oldPrice && oldPrice > price) {
    calculatedDiscount = Math.round(((oldPrice - price) / oldPrice) * 100);
    if (sheetDiscount > 0 && Math.abs(calculatedDiscount - sheetDiscount) > 1) {
        console.warn(`Discount in sheet (${sheetDiscount}%) doesn't match 
            calculation based on Rs.${price} vs Rs.${oldPrice}. 
            Updated to ${calculatedDiscount}%.`);
    }
}
```

Layer 2: Display Logic (GLOBAL_DISCOUNT Override)
------------------------------------------------
When rendering products, the system checks if GLOBAL_DISCOUNT is greater 
than zero. If set, it completely overrides both the displayed discount 
percentage and the current price, applying the global discount to all products.

Display Priority:
----------------
1. GLOBAL_DISCOUNT > 0: All products use global discount
2. GLOBAL_DISCOUNT = 0: Each product uses calculated discount

Affected Functions:
-------------------
- renderProducts() - Product grid on collections page
- renderHomeProducts() - Featured products on home page
- renderProductDetail() - Single product detail page
- renderFeaturedProducts() - Featured products section
- handleWhatsAppClick() - WhatsApp order message

Product Object Structure:
------------------------
| Field             | Type   | Description                              |
|-------------------|--------|------------------------------------------|
| price             | number | Current selling price                    |
| oldPrice          | number | Original price (can be null)              |
| discount          | number | Calculated discount % (displayed)        |
| originalDiscount  | number | Sheet discount % (for reference only)    |

Badge Colors by Discount Level:
-------------------------------
| Level   | Threshold    | Color     | Hex Code  |
|---------|--------------|-----------|-----------|
| SUPER   | >= 50%       | Green     | #27ae60   |
| HIGH    | >= 30%       | Red       | #e74c3c   |

Search Algorithm:
----------------
```javascript
function matchesSearch(product, query) {
    const q = query.toLowerCase();
    return (
        product.name.toLowerCase().includes(q) ||
        product.category.toLowerCase().includes(q) ||
        (product.description && product.description.toLowerCase().includes(q)) ||
        (product.brand && product.brand.toLowerCase().includes(q)) ||
        (product.colors && product.colors.some(c => c.toLowerCase().includes(q)))
    );
}
```
