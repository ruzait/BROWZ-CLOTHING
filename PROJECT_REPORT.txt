                          1. EXECUTIVE SUMMARY
================================================================================

Browz Clothing is a premium e-commerce website for a clothing and textile 
showroom in Addalaichenai, Sri Lanka. The platform enables the owner to 
manage products and promotional offers through Google Sheets without requiring
technical knowledge or code changes.

Key Achievements:
- Fully responsive design for all devices
- Zero-dependency data management via Google Sheets
- Integrated WhatsApp ordering system
- Professional SEO implementation
- High-performance lazy loading

================================================================================
                          12. OWNER MANUAL
================================================================================

Adding New Products:
--------------------
1. Open your Products Google Sheet
2. Add a new row with product information
3. Ensure "Product Name" and "Price" columns are filled
4. Add image URLs if available
5. Save the sheet
6. Wait 5 minutes or clear browser cache
7. Refresh the website

Adding Promotional Offers:
--------------------------
1. Open your Offers Google Sheet
2. Add a new row with offer details
3. Set "is_active" to "Yes"
4. Ensure "background_image" has a valid URL
5. Set "end_date" for the offer expiration
6. Save the sheet
7. Refresh the website immediately

Product Code Format:
--------------------
- Prefix: BC (configurable in main.js)
- Format: BC-0001, BC-0002, etc.
- Custom codes can be set in the sheet

Image Requirements:
-------------------
- Recommended size: 800x1000 pixels (4:5 ratio)
- Formats: JPG, PNG, WebP
- For Google Drive: Share > Anyone with link
- Convert sharing link to direct URL or use thumbnail format

Updating Prices:
----------------
1. Find the product row in Google Sheet
2. Update the "Price" column
3. Optionally update "Old Price" for discount display
4. Discount is auto-calculated if Old Price exists
5. Refresh website

================================================================================
                         13. TROUBLESHOOTING
================================================================================

Products Not Loading:
---------------------
1. Check browser console (F12) for errors
2. Verify Google Sheet is published (File > Share > Publish to web)
3. Check if CSV URL is accessible
4. Clear browser cache and retry
5. Verify column names match exactly

Images Not Displaying:
----------------------
1. Verify image URLs are public
2. For Google Drive, ensure sharing is enabled
3. Check URL format (https:// required)
4. Images may take time to load on first visit

Offers Not Showing:
-------------------
1. Check is_active column is "Yes"
2. Verify end_date is in the future
3. Ensure background_image URL is valid
4. Published URL may take time to update

WhatsApp Not Working:
---------------------
1. Verify WhatsApp number in CONFIG
2. Check browser allows opening WhatsApp links
3. Mobile devices may require WhatsApp app installed

Cache Issues:
-------------
1. Hard refresh: Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)
2. Clear browser cache
3. Incognito mode test
