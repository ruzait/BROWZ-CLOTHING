1. PROJECT OVERVIEW
--------------------------------------------------------------------------------
Mr. Nila Tex (Browz Clothing) is a premium clothing and textile showroom website 
based in Addalaichenai, Sri Lanka. The website showcases products for men, women, 
kids, and offers quality fabrics and textiles. Features include product 
browsing, category filtering, price range slider, search functionality, 
direct WhatsApp ordering, product detail pages, and an owner-friendly offers 
carousel system powered by Google Sheets.

Website URL: https://browzclothing.netlify.app

9. DISCOUNT SYSTEM LOGIC
--------------------------------------------------------------------------------
The discount system uses a priority-based approach:

Priority Order:
1. GLOBAL_DISCOUNT (from CONFIG) - Applied to ALL products if set > 0
2. Calculated Discount - Calculated from Old Price + Price when Old Price exists
3. Sheet Discount - Value from 'Discount %' column (used only if no Old Price)
4. No Discount - Default when neither is provided

Discount Calculation Formula:
    calculatedDiscount = (1 - Price / OldPrice) × 100

Badge Colors by Discount Level:
- SUPER (≥50%): Green (#27ae60)
- HIGH (≥30%): Red (#e74c3c)
- MEDIUM (≥15%): Orange (#e67e22)
- Default (<15%): Gold (#c9a959)

10. SECURITY IMPLEMENTATIONS
--------------------------------------------------------------------------------
XSS Protection:
- escapeHtml() function for all user-generated content
- Applied to: product names, brands, badges, item codes, descriptions
- WhatsApp messages use escaped content
- Form submissions use escaped user input

Form Validation:
- Email format validation (regex)
- Phone number validation (regex)
- Required field validation
- Focus on invalid field on error

URL Validation:
- isValidUrl() function checks URL format
- Requires http:// or https:// protocol
- Used for image URLs and Google Drive links

Null Safety:
- DOM element null checks added to all init functions
- Prevents errors on pages where elements don't exist
- initNavbar(), initSearch() include early return checks

11. IMAGE HANDLING SYSTEM
--------------------------------------------------------------------------------
Smart Image Display Logic (getProductImageHTML function):

1. Both images exist & different → Flip effect enabled
2. Only front image → Single image (no flip effect)
3. No images or load error → Placeholder SVG

Key Features:
- Automatic fallback to placeholder on image load failure
- Consistent aspect ratio (4:5) maintained
- Lazy loading for performance
- onerror handler for runtime image errors

Google Drive Link Conversion:
- Converts Google Drive sharing links to thumbnail URLs
- Format: https://drive.google.com/thumbnail?id=FILE_ID&sz=w800

14. OFFERS CAROUSEL BEHAVIOR
--------------------------------------------------------------------------------
Flow:
1. Page loads → Fetch offers from Google Sheet
2. Filter active offers (is_active = "Yes") with valid URLs
3. Sort by end_date (nearest first)
4. Display first offer with countdown
5. When countdown reaches 0 → Auto-switch to next offer
6. Hover over carousel → Show navigation arrows
7. Click arrow/dot → Manual navigation (resets auto-timer)
8. Loop back to first offer after last

Auto-Advance:
- Default: 8 seconds between slides
- Resets when user manually navigates
- Continues indefinitely (loops)

Mobile Behavior:
- Arrow buttons hidden on small screens
- Swipe gesture enabled for navigation
- Smooth scroll-snap behavior

## DESIGN SPECIFICATIONS

### Design Concept
Mr. Nila Tex is a distinguished clothing and textile establishment in Addalaichenai, 
Sri Lanka, offering premium fashion for the entire family. The website embodies 
**luxury without pretension** - a digital flagship store that feels like walking 
into an upscale boutique with warm Sri Lankan hospitality.

### Design Language

#### Aesthetic Direction
**Elegant Minimalism meets Sri Lankan Warmth** - Clean lines, generous whitespace, 
and sophisticated photography create a gallery-like experience, while gold accents 
and subtle textures add warmth and premium feel.

#### Color Palette
| Color | Hex | Usage |

| Primary Black | #1a1a1a | Sophisticated, authoritative |

#### Typography
| Element | Font | Style |
|---------|------|-------|
| Headlines | Playfair Display (serif) | Elegant, editorial feel |
| Body & UI | Poppins (sans-serif) | Modern, readable, friendly |

#### Spatial System
- Base unit: 8px
- Section padding: 80px-120px vertical
- Card gaps: 24px
- Maximum content width: 1400px
- Mobile padding: 20px

#### Motion Philosophy
| Animation Type | Duration | Effect |
|----------------|----------|--------|

### Adding/Editing Offers in Google Sheets

1. Open Google Sheet:
   https://docs.google.com/spreadsheets/d/1yr2_FNh55z2ryLj8j6NuB5E1aA8dk6acSZXgra6Fzms/edit

2. Go to "offerssheet" tab

3. Add or edit rows with these columns:
   - Column A: background_image (full URL like https://example.com/image.jpg)
   - Column B: badge_text (e.g., "Ramadan Offer")
   - Column C: title (e.g., "Ramadan Special Sale")
   - Column D: subtitle (e.g., "Special 30% OFF for your family!")
   - Column E: end_date (e.g., 2026-04-30T23:59:59)
   - Column F: is_active (Yes or No)

4. Save the sheet (auto-saves)

5. Refresh the website - changes appear automatically!

### Tips:
- Use full URLs (https://...) for background images
- Use <span class="highlight">text</span> in subtitle for gold highlighting
- Set is_active to "No" to temporarily hide an offer
- Offers auto-sort by end_date (nearest first)

### Mobile Notes:
- On mobile, you can swipe left/right to navigate offers
- Arrow buttons are hidden on small screens
